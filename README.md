# Melanoma_ICS
ICS model code and figure data

The melanoma ICS model is a customized version of SPPARKS (developed and distributed by Sandia National Laboratories). Here one can find the customized cpp files to build the ICS model with SPPARKS. To build the ICS SPPARKS executable, first download SPPARKS from https://sjplimp.github.io//download.html.
